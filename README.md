# Decopon - LINEミニアプリ

カーボンクレジットをクーポンに交換できるLINEミニアプリのプロトタイプです。

## ✨ 機能

- **🔄 スワイプ操作** - iPhoneライクなスワイプでクーポン引き換え・使用
- **🎫 クーポン管理** - 利用可能・引き換え可能・使用済みの状態管理
- **📱 ワンタイムバーコード** - セキュアな使い捨てバーコード生成
- **💚 LINEライクUI** - 本物のLINEアプリそっくりのデザイン
- **📊 リアルタイム残高** - クレジット残高の即座反映
- **📋 申請フォーム** - Jクレジット申請機能

## 🚀 技術スタック

- **Next.js 14** - App Router、TypeScript
- **Framer Motion** - スムーズなアニメーション
- **Tailwind CSS** - レスポンシブデザイン
- **Lucide React** - 美しいアイコン

## 🎯 主要コンポーネント

### 1. スワイプ操作（SwipeToConfirm）
iPhone標準の操作感を再現したスワイプ確定機能

### 2. LINEチャットUI
- リアルタイムメッセージ
- リッチメニュー（3分割レイアウト）
- タイピングインジケーター

### 3. クーポンシステム
- 引き換え可能 → 利用可能 → 使用済み
- 詳細モーダル表示
- バーコード生成（1回限り）

### 4. 申請フォーム
- 太陽光発電設備情報入力
- バリデーション付きフォーム
- 申請完了フロー

## 🔧 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番サーバー起動
npm start
```

## 📱 動作確認

1. ブラウザでアプリを開く
2. iPhoneフレーム内でLINEミニアプリが表示
3. リッチメニューから各機能にアクセス
4. スワイプ操作でクーポン引き換え・使用を体験
